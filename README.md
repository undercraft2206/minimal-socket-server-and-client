# minimal-socket-server-and-client
This is a minimal socket server and client to send a test message


# ⚠️ Attention ⚠️


Work only on linux, not on windows and macos !


There is juste an avertissement while compiling server.c 🐛


# to compile it

using gcc


```bash
gcc client.c -o client
````


using clang


```bash
clang client.c -o client
````
